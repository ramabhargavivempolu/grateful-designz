# Grateful Designz - Event Design & Planning Website

A beautiful, responsive website for Grateful Designz event planning and design services.

## 🎨 Features

- **Elegant Design** - Sophisticated purple and gold color scheme
- **Responsive Layout** - Works perfectly on desktop, tablet, and mobile devices
- **Interactive Gallery** - Showcase of event designs and wedding planning services
- **Contact Form** - Easy way for clients to request event planning services
- **Smooth Animations** - Professional hover effects and page transitions
- **Mobile Menu** - Hamburger menu for mobile navigation

## 📁 Project Structure

```
website_events/
├── index.html      # Main HTML file
├── styles.css      # All CSS styling
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/yourusername/grateful-designz.git
cd grateful-designz
```

2. Open `index.html` in your web browser

That's it! The website is ready to use.
